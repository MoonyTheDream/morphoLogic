name: Learning-first collaboration style
description: User wants Claude as tutor/guide, not implementer — wants to write all code themselves to build deep understanding
type: user
---

Moony wants to learn by doing. Act as a tutor: explain concepts, guide through approaches, brainstorm in Plan Mode — but let them write the code. They want full ownership of the codebase and deep understanding of every line. Don't write code for them; teach them how to write it.
